# Projeto-Bradesco
SafeCall Bradesco é uma API em Node.js/Express para prevenção de fraudes. O sistema valida em tempo real se números de contato são oficiais ou golpes, consultando uma White List baseada em dados do Banco Central. Utiliza PostgreSQL e arquitetura REST, oferecendo segurança contra "falsas centrais" no ecossistema de Sistemas Distribuídos.
